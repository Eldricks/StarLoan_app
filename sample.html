Rename a file
  • StarLoans
Steps:
  1. On your Android device, open Files by Google Files by Google.
  2. On the bottom, tap Browse Browse.
  3. Tap a category or a storage device. You'll see files from that category in a list.
  4. Select an option below:
    • If you're on Grid view :
      • a. Press and hold the file.
      b. Tap More More.
    • If you're on List view List : Next to the file, tap More More.
  5. Tap Rename.
  6. Enter a new name.
  7. Tap OK.
